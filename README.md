Anime Love Hub
🌸 Overview
Anime Love Hub is a collection of heartfelt anime-inspired love stories designed to warm hearts and spark imagination. Readers can explore short narratives, comment, and share their own creations.

✨ Features
Browse curated anime love stories

Categories: School Love, Fantasy Romance, Happy Endings, Saddest Stories

Interactive fan comments and likes

Option to write and publish your own story

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6)

Hosting: GitHub Pages

🚀 Getting Started
Clone the repository:

bash
git clone https://github.com/nehndiblessing/Anime-love-hub.git
Open index.html in your browser.

Explore stories or add your own.
